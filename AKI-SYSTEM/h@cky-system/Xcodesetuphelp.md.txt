Integration with Full AKI System
To connect with Ollama:
	1	Modify aki_backend.py to import your aki_simple.py
	2	Add Ollama calls in the backend
	3	Update AppState to handle streaming responses
Example Integration:

from aki_simple import process_query
import requests

def call_ollama(prompt, system_prompt):
    response = requests.post('http://localhost:11434/api/generate', 
        json={
            'model': 'deepseek',
            'prompt': prompt,
            'system': system_prompt
        })
    return response.json()
